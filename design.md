# Design System — Caio Augusto Ferreira, Psicólogo

## Identidade Visual
Baseada no cartão: fundo azul claro suave, detalhes em azul escuro navy. Cérebro tecnológico como logo central.
Sensação: tecnologia + saúde mental, modernidade, elegância, acolhimento.

## Cores
- `--blue-light: #B8D4E3` — fundo primário, hero, seções alternadas
- `--blue-mid: #7BAFC9` — bordas, separadores, elementos decorativos
- `--blue-dark: #1B3A6B` — títulos, textos principais, logo, CTAs
- `--blue-deep: #0F2547` — hover states, destaques
- `--white: #FFFFFF` — seções limpas, cards
- `--off-white: #F4F8FB` — fundo de seções alternadas
- `--text-body: #334155` — texto corrido
- `--text-muted: #64748B` — legendas, subtítulos

## Tipografia
- Display/Títulos: `Playfair Display` (Google Fonts) — elegância, autoridade
- Corpo/UI: `DM Sans` (Google Fonts) — moderno, legível, acessível
- Hierarquia: h1 3.5–4.5rem, h2 2.5rem, h3 1.5rem, body 1rem, line-height 1.7

## Espaçamento
- Seções: padding vertical 6–8rem
- Containers: max-width 1200px, padding horizontal 1.5rem
- Cards: padding 2rem, border-radius 1.5rem

## Estilo
- Minimalista sofisticado
- Bordas arredondadas generosas
- Sombras suaves (box-shadow com azul)
- Animações de entrada suaves (fade + translateY)
- Espaço em branco generoso
- Sem gradients agressivos — usar azul claro flat
- Ícones SVG minimalistas, stroke style

## Anti-patterns
- Sem roxo, sem verde
- Sem cards genéricos quadrados
- Sem aparência de clínica comum
- Sem fontes Inter/Roboto
